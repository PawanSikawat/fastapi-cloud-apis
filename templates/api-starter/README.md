# API Starter Template

Copy this directory into `projects/<project-name>/` when starting a new API.

## Rename These First

- directory name: `api-starter` -> your project slug
- package name: `template_api` -> your Python package name
- project metadata in `pyproject.toml`
- titles, descriptions, and README copy

## Included Baseline

- FastAPI Cloud-compatible flat project layout
- shared middleware wiring
- cookie-based browser UI auth flow
- one sample API endpoint
- one sample service
- templates for login and a simple dashboard
- tests with SQLite + fake Redis
